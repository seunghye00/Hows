## front-Architecture
### folder
- api :
  - API 사용 함수 작성
  - axios 데이터 통신 관리 폴더 
  - 해당기능.js 파일 만들어서 관리
- commons
  - 공통 함수 관리
- config
  - 설정 변수 또는 함수 만들어서 보관
- components
  - 공통 컴포넌트 관리
- pages
  - 페이지 컴포넌트 관리
- store
  - zustand 전역 상태관리
